# Support

## Fastest way

- Use the in-app `Report` button.

## GitHub support

- Open a GitHub Issue for bugs and feature requests:
  - [New bug report](https://github.com/playlistseeker/playlistseeker/issues/new?template=bug_report.yml)
  - [New feature request](https://github.com/playlistseeker/playlistseeker/issues/new?template=feature_request.yml)

## Include in every report

- App version (for example: `v1.1.0`)
- macOS version
- What you expected
- What actually happened
- Steps to reproduce

## Scope

This repository is a release/support channel and is not a full open-source contribution workflow.
